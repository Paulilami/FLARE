Tech Stack
Node.js: Main language for implementing the protocol and communication logic.
UDP Sockets (dgram module): For implementing the low-level UDP communication.
Express.js: If a simple web-based dashboard for drone monitoring is needed.
WebSockets: For real-time communication and feedback, if required later.
Docker: For containerization of the protocol, making deployment easy.
Mocha/Chai: Testing framework to ensure each module behaves as expected.
Shell Scripting: For automating drone setup and configuration.
